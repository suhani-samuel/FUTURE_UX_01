# Design Rationale – Glow Salon Website Redesign

## Target Audience
Women and young professionals looking for beauty and spa services.

## Goal
To create a modern and conversion-focused salon website that encourages users to book appointments easily.

## UX Decisions
- Soft pink theme for premium salon branding
- Simple navigation for better usability
- Testimonials for trust building
- Service cards for easy understanding
- Contact form for lead generation
- Mobile responsive layout for accessibility

## Conversion Strategy
- Clear CTA buttons
- Contact information visibility
- Strong visual hierarchy
- Easy booking-focused flow

## Conclusion
The redesign improves usability, trust, and conversion opportunities while maintaining a clean and modern user experience.
