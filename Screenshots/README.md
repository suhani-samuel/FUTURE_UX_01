# FUTURE_UX_01
# Glow Salon - Local Service Business Website Redesign

## UI/UX Design Internship Task 1 – Future Interns

### Project Overview
This project is a high-conversion website redesign for a local beauty and spa salon business called "Glow Salon".

The goal of this redesign was to improve:
- User experience
- Lead generation
- Conversion optimization
- Mobile responsiveness
- Service clarity and navigation

---

## Designed Pages
- Homepage
- Service Detail Page
- Contact / Lead Generation Page
- Mobile Responsive Screen

---

## Key Features
- Clean and modern UI
- Strong CTA sections
- Service pricing cards
- Testimonials and trust elements
- Lead generation contact form
- Mobile-friendly layout
- Consistent branding and spacing

---

## Tools Used
- Figma
- Google Fonts
- Coolors
- GitHub

---

## UX & Conversion Decisions
- Large CTA buttons improve booking visibility
- Testimonials build customer trust
- Simple navigation improves usability
- Soft color palette creates a premium salon feel
- Contact form placed prominently for lead generation
- Mobile layout improves accessibility for mobile users
